# carpoolcanarias Canarias
App moderna para compartir trayectos en las Islas Canarias. 

## Tecnologías
- **Framework:** Next.js (React)
- **Base de datos:** Supabase (Auth + Base de datos en tiempo real)
- **Mapas:** Mapbox para visualización de rutas.
- **Estilos:** Tailwind CSS.
- 
## 🛠️ Instalación local
1. Clona el repositorio.
2. Instala dependencias: `npm install`
3. Configura tus variables de entorno en un archivo `.env.local`.
4. Lanza el proyecto: `npm run dev`
